# 매개변수란?
- 함수가 사용할 값 받는 변수

```cpp
int add(int a, int b) //int a, int b 요놈들이 매개변수{

	return a + b;


}
```

일단 솔직하게 아직 용어도 어려운 부분이 있기 때문에 모르는 단어가 있으면 무조껀 다 기록을 하자!
모르는건 부끄러운 것이 아니다 모르는데도 부끄러워서 넘어가는게 진짜 부끄러운 것!

즉 매개 변수 = 함수 호출할 때 전달받는 값 저장하는 변수 
근데 여기서 실제 넣는 값 (10,20) 이런건 인자라고 한다 
함수 = 기계
매게변수 = 재료 넣는 곳

## 오버로딩 특징

**오버로딩이란? 같은 이름의 함수를 여러 개 만들어서 상황에 따라 다르게 쓰는 것**
**함수 오버로딩은 "함수 이름"이 아니라 "매개변수"로 구분**
**함수 오버로딩은 "매개변수의 개수나 타입"이 다르면 가능하다**
**리턴 타입은 오버로딩 구분 기준이 아니다**

## 왜 쓰는가?
**같은 기능은 같은 이름으로 묶기 위해**
- 비슷한 기능을 하는 함수들은 하나의 이름으로 묶어서 코드 관리와 사용을 쉽게 하려고 사용

## 참조
void = 결과를 돌려줄 필요 없이 실행만 하면 되는 함수
값을 return 할려면 void 쓰면 안된다


## 언제 쓰는지 파악
return 있음 -> 결과 필요
void -> 결과 필요 없음

return -> 값 계산, 값 전달 -> 알려준다
void -> 출력/실행, 상태 변경 -> 보여준다

## 디폴트 매개변수

** 함수에 값을 안 넣으면 기본값을 자동으로 넣어주는 기능
- 값을 안 넣으면 기본값 사용(예시 : int crit = 0;)
- 값을 넣으면 기본값 무시(지금 기본값은 0이지만 값을 넣으면 기본값을 무시하고 들어감)
- 뒤에서부터만 설정 가능

결론 : 디폴트 매개변수는 '반복'적으로 같은 값을 넣는 것을 줄여서 코드를 간결 관리 쉽게 만듬
