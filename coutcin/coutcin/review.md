## 헷깔린 것
똑같은 변수에 입력값을 두번 입력 한다면?
그럼 변수가 두개일 때 입력 값 을 두번 입력한다면?
예를 들어 이런 상황

```cpp
int age;
std::cin >> age;
std::cin >> age;
```

```cpp
int a,b;
std::cin >> a >> b;
```

## 내가 생각한것
- 두개의 값을 받는다고 생각 
- a 값을 입력해도 b값이 출력 될 것이다

## 생각 바꾼 점
- 두 번 입력 받는건 맞지만 변수가 같아서 같은 변수에 저장됨 값이 두개 저장되는게 아니라 하나가 계속 덮어쓰기 된다
- 이건 변수가 달라서 공간이 다 다르므로 순서대로 저장된다
