# 입출력 함수
사용자와 컴퓨터가 소통 할 수 있게 도와주는 도구이다

-cout : 출력
-cin : 입력

## 왜 쓰는가??
**컴퓨터 안의 값을 보고, 사용자한테 값을 받기 위해서 사용한다**
- 컴퓨터 안의 값은 사용자가 직접 못보기 때문에 출력이 필요함
- 사용자가 필요로한 입력값을 컴퓨터도 알아야하기 때문에 입력이 필요함
- cout이 없으면 변수 값이 눈에 보이지 않음

# #std는 뭔가?
std::cout 이란건 cout이 std라는 공간 안에 있는 것이다
- std = 표준 라이브러리 공간
- cout = 그 안에 있는 기능
- 즉 말로 표현하면 std안에 있는 cout을 쓰겠다

**cin 주의 사항**
cin은 공백(스페이스, 엔터)을 기준으로 값을 나너서 입력 받는다
cin >> a >> b 는 값을 순서대로 두 개 받아서 각각 저장한다
